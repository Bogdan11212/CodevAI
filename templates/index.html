Query
Write a function to check if a string is a palindrome
AI Thinking

1. A palindrome reads the same forward and backward

2. We need to compare characters from both ends

3. Ignoring case and non-alphanumeric characters would be good

Solution 
def is_palindrome(text):
    # Remove non-alphanumeric and convert to lowercase
    clean_text = ''.join(c.lower() for c in text if c.isalnum())
    
    # Compare the string with its reverse
    return clean_text == clean_text[::-1]

def fibonacci(n):
    if n <= 0:
        return 0
    elif n == 1:
        return 1
    else:
        return fibonacci(n-1) + fibonacci(n-2)

# CodevAI thinking process:
# 1. This is a recursive implementation of Fibonacci
# 2. It works correctly but has exponential time complexity
# 3. We can optimize with memoization to avoid repeated calculations
# 4. Let's implement an optimized version with O(n) time complexity

def fibonacci_optimized(n, memo={}):
    if n in memo:
        return memo[n]
    if n <= 0:
        return 0
    elif n == 1:
        return 1
    else:
        memo[n] = fibonacci_optimized(n-1, memo) + fibonacci_optimized(n-2, memo)
        return memo[n]
